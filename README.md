# 🎵 Spotify Web Player Clone

A lightweight, responsive web application that replicates the core UI and audio playback functionality of Spotify. Built entirely from scratch using pure **HTML5**, **CSS3**, and **Vanilla JavaScript** — no external libraries or heavy frameworks required.

---

## ✨ Features

- **Interactive Audio Player:** Play, pause, skip, and scrub through tracks with real-time progress updates.
- **Dynamic Playlists & Queue:** Browse tracks, view albums, and select songs directly from structured lists.
- **Volume & Mute Controls:** Fine-tune audio levels seamlessly using interactive range sliders.
- **Responsive Design:** Optimized layout for desktop, tablet, and mobile screens using CSS Grid and Flexbox.
- **Custom UI Components:** Recreates Spotify’s iconic dark theme, sidebar navigation, bottom player controls, and hover animations.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic layout (player bar, sidebar, main content area).
- **CSS3:** Custom styles, Flexbox, Grid, transitions, and responsive media queries.
- **JavaScript (ES6+):** DOM manipulation, HTML5 Audio API integration, event handling, and state management.
